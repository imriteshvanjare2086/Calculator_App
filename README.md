# Calculator Web App

A simple and responsive calculator application built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and includes additional functional buttons like AC (All Clear), DEL (Delete), and Percentage.

## Features

*+ Addition*

*- Subtraction*
 
*x Multiplication* 

*/ Division*

*% Percentage*

*AC (Clear all input)*

*DEL (Delete last digit)*

*Double zero (00) button*

*Clean and modern UI*

## Technologies Used

- HTML5 : Structure of the calculator

- CSS : Styling and layout design

- JavaScript : Calculator logic and interactivity


## UI Design Highlights

- Centered calculator layout

- Circular buttons with shadow effects

- Color-coded operators

- Smooth input transition effects
